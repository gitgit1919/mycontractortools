# Affiliate Program Tracking

Central log of every affiliate program, its status, and the tracking URL to swap into the site once approved. Update this file as approvals come in. When a batch of URLs is ready, Claude can do a single bulk swap across the site.

**Last updated:** 2026-04-10

---

## How to use this file

1. When you submit an application, change the status from `Not applied` to `Pending` and fill in the `Applied` date.
2. When you get approved, change status to `Approved`, paste the tracking URL in the `Tracking URL` column, and add the commission rate if different from the assumed rate.
3. When you want Claude to update the site, ask: "Swap the approved affiliate URLs into the site from affiliate-tracking.md" — it will find every placeholder that matches an approved program and replace it with the real URL.
4. Keep notes on each row if a program has quirks (e.g., "rejected, reapply in 6 months" or "use deep link for HVAC page").

**Status values:** `Not applied` · `Pending` · `Approved` · `Rejected` · `Active` (earning)
